## 1.0.0 - 2026-05-25

### 新功能
- 签到倒计时模式 — 替换时间窗口，新增"开始签到"按钮，支持 30 分钟倒计时
- 班级归档功能 — 教师可归档/恢复班级
- 历史批次列表分页 — 每页10条，支持上下翻页
- 学生签到搜索支持拼音首字母和全拼（输入 `csy` 匹配陈思源）
- 学生签到增加 IP 限制 — 同一 IP 每节课程只能签到一次
- 学生签到成功后自动清除其所带标签
- 签到页支持回车提交 — 已选学生直接签到、仅一条自动签到
- 预设标签改为数据库管理 — 管理员可增删改预设标签
- 预设标签变更 SSE 实时推送 + 级联更新学生标签
- 座位表支持标签管理 — 点击左侧 roster 项目打开标签弹窗
- 座位表新增"上次座位"按钮，支持查看上节课座位及变动对比
- 座位表比对区分换座位和新签到（黄色标变动，蓝色标新增）
- 座位表名单显示行政班 + 历史批次座位显示学生标签
- 管理员预设标签管理 + 看板签到中状态
- 学生管理新增手动添加学生功能

### 修复
- 修复多个安全漏洞 — CSRF、XSS、Excel注入、时序攻击等
- 签到并发竞态防护 + 修复倒计时过期后不自动归档
- SSE 返回 ERR_INVALID_HTTP_RESPONSE 修复
- 审计问题修复 — 分页、缓存、速率限制等
- 全局限流调整至 10000/min，解决教室多浏览器触发429问题
- 历史批次模态框固定高度，分页切换不抖动
- 修复 analytics 中 `$queryRaw` 返回 BigInt 的混合类型错误
- 修复多处前端空指针崩溃 + NaN 检查
- 标签编辑弹窗居中 + 重名校验避免唯一约束冲突
- 允许班级重名 + 学生转班标签自动清理
- 签到名空格匹配修复 + studentIp 迁移避免 IP 检查被绕过

### 性能优化
- 消除 N+1 查询 — 合并重复查询、数据库索引优化
- 大事务拆分 + bcrypt 并行哈希
- SSE 缓存 + 事务内联优化

### 重构
- 清理 SignInConfig 死字段 startTime/endTime 及冗余无用代码
- 统一签到标签清除逻辑（仅清除自定义标签，保留预设标签）

### 样式优化
- 签到页严格按设计系统重构 — 上下布局、主题切换、时钟组件
- 统一管理端四个页面样式 — 抽取公共 admin.css
- 签到按钮、标签、排版等细节优化
